# エンジニア社内勉強会 資料リポジトリ

## 概要

本リポジトリは、エンジニアチームの社内勉強会の資料を一元管理するものです。

## 運営方針

### 🎯 Goal
**業務時間内（30分）で完結し、知見を持ち帰る。**

### 📏 Policy
* **宿題・事前準備なし（No Homework）**: 参加者は事前準備不要です。
* **心理的安全性**: 「知らない」は恥ではない。質問歓迎の環境を作ります。

### 🎓 Target
開発チーム全メンバー（社長も閲覧可能）
