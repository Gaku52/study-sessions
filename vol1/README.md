# 第1回 エンジニア社内勉強会 資料

## 概要

**開催日：2026/01/21**

本ディレクトリには、第1回エンジニア社内勉強会の資料が格納されています。

## 運営方針

### 🎯 Goal
**業務時間内（30分）で完結し、知見を持ち帰る。**

### 📏 Policy
* **宿題・事前準備なし（No Homework）**: 参加者は事前準備不要です。
* **心理的安全性**: 「知らない」は恥ではない。質問歓迎の環境を作ります。

### 🎓 Target
開発チーム全メンバー（社長も閲覧可能）

## 使い方

`slides/` ディレクトリ内の3つの案から、当日のチーム状況に合わせて最適な形式を選択して実施します。

### 資料一覧

| ファイル名 | 形式 | 概要 |
|-----------|------|------|
| `01_hybrid.md` | ハイブリッド形式 | 個人学習 + テーマが被った人はチームで会話OK |
| `02_leader_share.md` | リーダー共有形式 | Gakuが今日のインプットを共有する講義型 |
| `03_refactoring.md` | リファクタリング形式 | サンプルコードを題材に議論する |
