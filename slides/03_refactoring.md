# 第1回 エンジニア勉強会

**テーマ：コード・リファクタリング検討会**
〜そのコード、もっと良くできませんか？〜

---

## 本日のゴール

### 🎯 ゴール
**「他人の視点を取り入れて、コード品質の基準を上げる」**

今日は、短いサンプルコード（お題）を1つ出します。
「自分ならこう書く」「ここがバグの温床になりそう」という意見を出し合い、チームとしての**ベストプラクティス**を探りましょう。

---

## 本日のお題（The Code）

**この処理、どう思いますか？**

```javascript
// 例: 条件分岐が複雑な関数
function checkUser(user) {
  if (user) {
    if (user.age > 20) {
      if (user.isActive) {
        return true;
      }
    }
  }
  return false;
}
```

---

## Thinking Time（5分）

**各自、以下の視点で修正案を考えてください**

* **可読性**（読みやすいか？）
* **保守性**（変更に強いか？）
* **パフォーマンス**（無駄がないか？）

※ チャットに修正後のコードを貼る準備をしてください。

---

## Discussion（15分）

**みんなの修正案・指摘をシェア**

* 「私はこう直しました」（早期リターン、ガード節など）
* 「ここが気になりました」（型定義など）
